Data Visualization Using d3.js

2015-03-03 #d3.js #visualization

FindNewAlbums

A simple implementation in d3.js.

Based on Clustered Force Layout III.

Data scrarched from RYM. Using top 18 albums listed in RYM, find lists including these albums respectively, and get albums inside these lists, then map relating albums with the original albums to show the albums similar with the original ones.
